# 📊 Data Analyst Job Market Intelligence – SQL Analysis Summary

## 📌 Project Context

This project analyzes a structured PostgreSQL job postings database developed as part of Luke Barousse’s SQL course.  

The dataset includes:

- Job titles
- Work types (remote / on-site)
- Locations
- Required skills
- Company information
- Average yearly salaries

The objective of this analysis is to uncover **high-paying remote Data Analyst opportunities** and identify the **skills that maximize earning potential**.

---

# 1️⃣ Highest Paying Remote Data Analyst Roles

## 🎯 Objective
Identify the top-paying remote Data Analyst positions with verified salary data.

<img src="images/top_paying_remote_roles.png" width="900">

### 🔎 Key Insights

- Executive-level roles (Director / Associate Director) dominate top salary tiers.
- Fully remote positions (“Anywhere”) can exceed $250K annually.
- Large tech and enterprise companies lead compensation levels.
- Salary transparency significantly narrows the analysis to verified market value roles.

This section highlights where Data Analysts can maximize compensation while working remotely.

---

# 2️⃣ Skills Required for the Highest Paying Roles

## 🎯 Objective
Extract required skills from the top 10 highest-paying remote roles.

<img src="images/top_paying_roles_skills.png" width="900">

### 🔎 Key Insights

- SQL and Python consistently appear in top-tier job listings.
- Cloud and data platform tools (Azure, AWS, Databricks) are strongly represented.
- Modern analytics stack tools (Pandas, PySpark, Jupyter) are highly demanded.
- High-paying roles expect a hybrid of technical depth and infrastructure knowledge.

This reveals the technical stack commonly associated with premium compensation.

---

# 3️⃣ Top Paying Skills in Remote Data Analyst Roles

## 🎯 Objective
Determine which individual skills command the highest average salary.

<img src="images/top_paying_skills.png" width="900">

### 🔎 Key Insights

- Distributed computing (PySpark) commands the highest average salary.
- DevOps and infrastructure-related tools (GitLab, Couchbase, Datarobot) show strong salary premiums.
- Advanced analytics and ML-related tooling correlates with higher compensation bands.

This suggests that advanced data engineering and ML-aligned skills elevate salary potential.

---

# 4️⃣ Optimal Skills: Demand vs Salary Balance

## 🎯 Objective
Identify skills that combine **high market demand** with **strong salary potential**.

<img src="images/optimal_skills.png" width="900">

### 🔎 Key Insights

- SQL, Python, and Excel dominate demand volume.
- Tableau and Power BI show strong demand but moderate salary range.
- Highly specialized tools may offer higher salary but lower overall demand.
- The best career strategy balances both **market demand** and **salary ceiling**.

This section provides a practical roadmap for career prioritization.

---

# 🧠 Strategic Conclusions

### 💼 For Job Seekers
- Master SQL and Python as foundational skills.
- Expand into cloud, distributed computing, and ML tooling.
- Remote opportunities can match or exceed on-site compensation.

### 🏢 For Organizations
- High compensation correlates with hybrid analytics + engineering skill sets.
- Remote hiring broadens access to high-value talent.
